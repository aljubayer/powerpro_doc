Requirements

You will need the following requirements to customize this Theme.

  1. Wordpress ( 4.7 ~ 5.0.2 )
  2. PHP ( 5.6 or 7.2 )

Plugins

  1. Codexin Core
  2. Elementor Page Builder
  3. Contact Form 7
  4. Kirki
  5. Smart Slider 3

These plugins can be installed and activated via PowerPro theme activataion.

Installation

After purchasing the PowerPro theme, download it from your themeforest account and save the zip package in your local computer.

This download package consist of documentation, license and a zipped file. Now unpack the zipped file entirely in your local computer.

Installation Of Theme

Using Wordpress Theme Manager:

After Installtion of your wordpress application or previously installed wordpess, goto Appearance > Themes

and upload the powerpro.zip file.

Once the upload has been finished, activate the theme.

Using FTP Upload

Upload unzipped powerpro folder to the /wp-content/themes/ folder in the WordPress installation on your hosting server.

Install And activate theme required plugins:

After activating the theme, admin notices will show the listing of required and recommended plugins as below.

Now Install the required and recommended plugins using admin notces clicking Begin intalling plugins.
